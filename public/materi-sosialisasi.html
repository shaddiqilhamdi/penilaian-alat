Apa itu Aplikasi Ini?

Objektif Utama:
"Memastikan setiap pekerja di lapangan terlindungi dengan APD & peralatan kerja yang layak, lengkap, dan berfungsi ‚Äî melalui sistem monitoring digital real-time, sehingga seluruh ketidaksesuaian dapat terdeteksi dini dan ditindaklanjuti segera."

Aplikasi Penilaian Alat Kerja adalah sistem digital untuk:

  • Deteksi Dini ‚Äî Mengetahui secara cepat jika ada alat yang rusak, hilang, atau tidak layak pakai
  • Monitoring Real-Time ‚Äî Memantau kondisi APD & peralatan kerja setiap regu/personil secara langsung
  • Tindak Lanjut Cepat ‚Äî Data tercatat langsung bisa ditindaklanjuti untuk penggantian atau perbaikan
  • Zero Incident Target ‚Äî Mendukung budaya K3 dengan memastikan tidak ada pekerja bekerja dengan peralatan tidak standar

"Pantau Alat, Lindungi Pekerja ‚Äî Real-Time, Setiap Saat"

Troubleshooting Umum

Masalah 1: Tidak Bisa Login
Gejala: Muncul pesan "Email atau password salah"
  • Pastikan CAPS LOCK tidak aktif
  • Periksa ejaan email
  • Coba klik "Lupa Password" untuk reset
  • Hubungi Admin jika masih gagal
Masalah 2: Halaman Error
Gejala: Halaman kosong atau error
  • Refresh halaman (F5)
  • Hapus cache browser (Ctrl+Shift+Delete)
  • Coba browser lain
  • Periksa koneksi internet
Masalah 3: Data Tidak Tersimpan
Gejala: Klik Simpan tapi tidak ada konfirmasi
  • Periksa koneksi internet
  • Pastikan semua field wajib sudah diisi
  • Jangan klik Simpan berulang kali
  • Tunggu hingga muncul notifikasi
Masalah 4: Dropdown Kosong
Gejala: Pilihan vendor/kendaraan/petugas tidak muncul
  • Penyebab: Data belum diinput oleh Admin
  • Solusi: Hubungi Admin UP3 untuk menambahkan data master

Pertanyaan Umum (FAQ)

Q: Apakah bisa input penilaian untuk tanggal kemarin?

A: Ya, bisa memilih tanggal sebelumnya saat input penilaian.

Q: Bagaimana jika ada alat yang tidak ada di daftar?

A: Hubungi Admin untuk menambahkan alat ke standar peralatan.

Q: Apakah data bisa diedit setelah disimpan?

A: Data yang sudah disimpan tidak bisa diedit untuk menjaga integritas.

Q: Bagaimana melihat riwayat penilaian?

A: Buka menu Penilaian ‚Üí Data Penilaian, lalu filter sesuai kebutuhan.

Q: Siapa yang bisa melihat data penilaian?

A:

  • UID Admin/User ‚Üí Semua data UP3
  • UP3 Admin/User ‚Üí Data UP3 masing-masing
  • Vendor K3 ‚Üí Data vendor sendiri
